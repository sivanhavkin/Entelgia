This directory contains project assets such as images, logos, icons, and other media files used in the Entelgia project.

## Purpose

The Assets folder is organized to store:
- Images and graphics
- Logo files
- Icons
- Screenshots
- Diagrams
- Other media resources

## Usage

Place all project-related media files in this directory to maintain a clean and organized repository structure.

## Web Research

As of v2.8.0, the Web Research Module may retrieve images or media referenced
on web pages but stores only extracted text, not binary media files.
Screenshots illustrating the web research pipeline output can be placed here.
