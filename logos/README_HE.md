# לוגו לפרויקט Entelgia - תיק עיצובים

## 🎨 הצעות עיצוב

יצרתי **5 אופציות לוגו מקצועיות** לפרויקט Entelgia. כל עיצוב מייצג היבט אחר של הארכיטקטורה הרב-סוכנית של המערכת.

### 📋 סקירת האופציות

#### אופציה 1: רשת מוחית (Brain Network)
- **נושא:** ארכיטקטורה עצבית ובינה מלאכותית
- **סגנון:** טכני, מדעי
- **צבעים:** סגול/אינדיגו
- **מתאים ל:** תיעוד טכני, הקשרים של AI/ML, כותרת GitHub

#### אופציה 2: מעגלי דיאלוג (Dialogue Circles)
- **נושא:** דיאלוג רב-סוכני ותודעה
- **סגנון:** קונספטואלי, פסיכולוגי
- **צבעים:** כחול/סגול/ורוד
- **מתאים ל:** מצגות, הסבר ארכיטקטורת המערכת, חומרים חינוכיים

#### אופציה 3: מוח גיאומטרי (Geometric Mind)
- **נושא:** פוטוריסטי ומודרני
- **סגנון:** סייברפאנק, טכנולוגי
- **צבעים:** תכלת/סגול/ורוד
- **מתאים ל:** אתרים בעיצוב כהה, מיתוג טכנולוגי מודרני, רשתות חברתיות

#### אופציה 4: E מינימליסטי (Minimal E)
- **נושא:** מקצועי ונקי
- **סגנון:** מינימליסטי, תאגידי
- **צבעים:** אינדיגו/סגול
- **מתאים ל:** הקשרים עסקיים, favicon, גדלים קטנים, שימוש רב-תכליתי

#### אופציה 5: שכבות זיכרון (Memory Layers)
- **נושא:** ארכיטקטורת זיכרון מתמשך
- **סגנון:** אבסטרקטי, שכבתי
- **צבעים:** תכלת עד ורוד
- **מתאים ל:** הדגשת תכונות ייחודיות, מאמרים טכניים, דיאגרמות ארכיטקטורה

### 🔍 כיצד לצפות בלוגואים

1. **תצוגה אינטראקטיבית:** פתח את `logos/index.html` בדפדפן לראות את כל הלוגואים עם וריאציות רקע
2. **קבצים בודדים:** פתח כל קובץ SVG ישירות בדפדפן או מציג תמונות
3. **תיעוד:** קרא את `logos/README.md` להסברים מפורטים

### 🎯 שלבים הבאים

1. **בחר את העיצוב האהוב עליך** מתוך 5 האופציות
2. **בקש שיפורים** (צבעים, פרופורציות, אלמנטים)
3. **צור וריאציות** (רקעים בהירים/כהים/שקופים, גדלים שונים)
4. **ייצוא פורמטים** (PNG, JPG, ICO favicon, PDF להדפסה)

### ✨ יתרונות הקבצים

כל הלוגואים נוצרו כ-**SVG (Scalable Vector Graphics)**:
- ✅ ניתנים להגדלה אינסופית ללא איבוד איכות
- ✅ גודל קובץ קטן
- ✅ קלים לעריכה והתאמה אישית
- ✅ ידידותיים לאינטרנט
- ✅ איכות מקצועית
