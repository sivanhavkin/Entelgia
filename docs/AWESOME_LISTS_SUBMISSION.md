### How to Submit

1. **Fork the Repository**
   ```bash

   # Click "Fork" button in top-right corner
   ```

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/awesome-agents.git
   cd awesome-agents
   ```

3. **Edit README.md**
   - Find the appropriate section (e.g., "Multi-Agent Systems" or "Agent Frameworks")
   - Add the entry in alphabetical order
   - Use the format: `- [Project Name](URL) - Brief description.`

4. **Commit Your Changes**
   ```bash
   git add README.md
   git commit -m "Add Entelgia - consciousness-inspired multi-agent AI architecture"
   git push origin main
   ```

5. **Create Pull Request**
   - Go to your forked repository on GitHub

## Project Highlights for Submissions

When describing Entelgia in submissions, emphasize:

### Key Features
- **Persistent Identity:** Maintains evolving state across sessions
- **Multi-Agent Architecture:** Three distinct agents (Socrates, Athena, Fixy)
- **Shared Memory:** Encrypted long-term memory system
- **Moral Regulation:** Emergent ethical behavior through agent interaction
- **Local LLM:** Runs on Ollama (privacy-focused, no cloud dependency)

### Technical Stack
- Python 3.10+
- Ollama for local LLM execution
- FastAPI for API endpoints
- Encrypted memory storage
- Comprehensive test suite (24+ tests)

### Use Cases
- Research on AI consciousness and identity
- Multi-agent dialogue systems
- Persistent AI assistants
- AI ethics and moral reasoning experiments

### Project Status
- **Status:** Research/Beta - Active Development
- **License:** MIT
- **Maintenance:** Active (2026)
- **Tests:** 24 passed, CI/CD enabled
