# Quick Guide: Submit Entelgia to awesome-agents

This is a streamlined guide to submit Entelgia to the primary awesome list.

## TL;DR - 5 Minute Process

### Step 1: Fork and Clone

1. Go to: https://github.com/l-aime/awesome-agents
2. Click "Fork" (top-right)
3. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/awesome-agents.git
   cd awesome-agents
   ```

### Step 2: Edit README.md

Find the appropriate section (e.g., "Multi-Agent Systems" or "Agent Frameworks") and add:

```markdown
- [Entelgia](https://github.com/sivanhavkin/Entelgia) - A consciousness-inspired multi-agent AI architecture that models persistent identity, internal conflict, and emergent moral regulation through shared long-term memory and structured dialogue.
```

**Important:** Add in alphabetical order within the section!

### Step 3: Commit and Push

```bash
git add README.md
git commit -m "Add Entelgia - consciousness-inspired multi-agent AI architecture"
git push origin main
```

3. Click the pencil icon (Edit this file)

7. Click "Create pull request"
